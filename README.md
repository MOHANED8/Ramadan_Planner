# Ramadan Planner Extension

## Running Locally

This project is a Progressive Web App (PWA) and can be run as a standard web application.

### Prerequisites

- Python 3.x (Recommended) or Node.js

### Instructions

1.  Open a terminal in this directory.
2.  Start a local HTTP server:
    - **Using Python:** `python -m http.server 8080`
    - **Using Node.js:** `npx http-server`
3.  Open your browser and navigate to: [http://localhost:8080](http://localhost:8080)

## Project Structure

- `index.html`: Main application entry point.
- `script.js`: Core application logic.
- `manifest.json`: Web App Manifest for PWA features.
- `sw.js`: Service Worker for offline capabilities.
